# Address-Book
Based on Nodejs and MongoDB Atlas to fetch, delete, update and edit data through respective api endpoints


### Installation and Setup Instruction:

First you need to have `git`, `node` and `npm` installed. You can find `git` [here](https://git-scm.com) and `node` [here](https://nodejs.org).

Now clone this repo, use `git clone https://github.com/Animesh-456/Address-Book.git`

Now use `npm install` to install all the required dependencies.

To start the app (in dev environment) use `npm run devStart` or you can use `npm run start` too.

Now open your browser of choice on your local machine and navigate to `localhost:3000`

### API endpoints implemented

◆ Add a new contact.<br>
◆ Fetch details of single contact.<br>
◆ Fetch the list of contacts with pagination.<br>
◆ Update the given contact.<br>
◆ Delete the given contact.<br>


### NOTE: - Everything is done in MongoDB Atlas
